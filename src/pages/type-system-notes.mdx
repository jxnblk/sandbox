
# Type System

Simple abstraction for creating base typographic styles

## Input
- scale
  - OR ratio for generative scales
- fontFamily
- fontWeight
- fontSize (p)
- lineHeight
- margin (based on space scale)

## Output
- fontSizes
- space scale
- styles (all markdown elements)
  - h1-h6
  - p, ol, ul, li
  - code, samp, pre, kbd
  - blockquote
  - hr, img,

## Implementations
- TypeSystemProvider
- Emotion global styles
- Raw CSS
